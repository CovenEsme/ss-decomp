# Necessary files

In this folder, place the extracted `main.dol` file. Then, create a `rels` folder and place the necessary rels inside.

Your folder structure should look like this:
```
├── orig
    ├── SOEU01.sha1
    └── SOEU01
        ├── main.dol
        ├── README.md
        └── rels
            ├── d_a_asura_bulletNP.rel
            ├── d_a_avater_bulletNP.rel
            ├── ...
```
